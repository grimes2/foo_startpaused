# foo_stopafteralbum
For `foobar2000` `v2` and newer: Activate stop after current, if TRACKNUMBER = TOTALTRACKS.

# Building from source
Requirements:
* https://www.foobar2000.org/SDK
* https://wtl.sourceforge.io/
